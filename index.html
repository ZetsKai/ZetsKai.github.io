Airplane Mode

All wireless communication, such as WI-FI and Bluetooth, will be turned off. Some types of wireless communication can still be switched on manually when the console is in Airplane Mode.

This can also be changed from Quick Settings (hold *).

Suspend Auto-Sleep While Playing Media Content

Prevents auto-sleep from activating while media content is playing.
If this option is disabled, the console will still go to sleep after four hours of media playback.

Maintain Internet Connection in Sleep Mode

If this option is enabled, downloads from your PC or smartphone will start right away, even when the console is in sleep mode. Disabling this option will reduce the console's energy consumption during sleep mode while connected to the Internet.

Wake When AC Adapter Is Disconnected

If this option is enabled, the console will wake up from sleep mode when it is removed from the dock.

Controller Vibration

The vibration feature will be enabled on supporting controllers during gameplay. Vibration for alarm notifications can be changed in Notifications => Alarm Notifications.

Pro Controller Wired Communication

If this option is enabled, the Nintendo Switch Pro Controller will communicate with the console via wired communication when connected to it using the USB charging cable. The NFC touchpoint on the Pro Controlller will be disabled while the controller is using wired communication.
